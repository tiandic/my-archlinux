# 我的arch Linux 配置
这个仓库的初衷是为了能够让我自己能清楚知道自己对`arch linux`做了哪些配置,以便迁移部分配置,并为打算要使用`arch linux`,且想要为之做出自定义配置的人提供一个参考

因此,每篇文章都会对所进行的操作进行一定的说明,以指南的形式编写这些文章,并在文末提供一个简单的自动化脚本,以便快速迁移该配置

另外,本仓库假设读者已经了解一定的Linux相关知识或者能够有搜寻这些知识的能力,因此,虽然每篇文章都具有十分详细的步骤,但相关知识的说明可能相对匮乏

由于本人在创建本仓库时也是第一次接触`arch linux`,所以我想本仓库应对一些`arch linux`新手具有一定的参考价值

如果你在寻找一个`arch linux`配置指南,那么可以看看本仓库,看看以前的新手是如何一步一步配置他的`arch linux`的

那么话不多说,让我们开启这趟心潮澎湃的旅程吧!
